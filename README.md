# Yinpeng Su Portfolio

A static, GitHub-ready personal portfolio website with English / Chinese / French language switching.

## Pages

- `index.html` — personal homepage with forest/deer visual identity, work style, about section, and public links.
- `cv.html` — technical research CV focused on research software, data analysis, simulation, and interactive visualization.
- `projects.html` — WorkLord Toolbox project page with variable explanations, an interactive demo, figure gallery, code architecture, and project reflection.

## Local preview

Open `index.html` directly in a browser, or use VS Code Live Server.

## GitHub Pages deployment

1. Create a public repository named `YinpengSu-Portfolio`.
2. Upload all files in this folder to the repository root.
3. Go to `Settings → Pages`.
4. Select `Deploy from a branch`.
5. Choose `main / root`.
6. Wait for GitHub Pages to publish the site.

Expected URL format:

```text
https://YinpengSu-Concordia.github.io/YinpengSu-Portfolio/
```

## Privacy

The public site keeps direct email and phone hidden. LinkedIn and GitHub are used as the public contact paths.
