# Final Public Site Notes

This version is prepared for public GitHub Pages / Netlify deployment.

Key choices:

- Email and phone are not displayed on the public site.
- LinkedIn and GitHub are used as public contact paths.
- The homepage is intentionally more personal: forest/deer visual identity, TRPG/dice note, and a clear but non-sales style.
- The CV page remains professional, with a small note at the bottom about seasonal park staff and cashier experience.
- The Projects page separates Mental Capacity from β and includes an interactive scenario interpretation button.
- The WorkLord web demo is explicitly described as a simplified explanation layer, while the full computation remains in the Python Toolbox.
